# data-analyst-portfolio
Análisis de +1000 ofertas laborales de 6 categorías (Business Analyst,  Cloud, Data Scientist, HR, Software Developer, UI/UX) en múltiples países.

## Herramientas:
- Excel (limpieza, fórmulas, gráficos, dashboard).

## Hallazgos principales:
- UI/UX lidera en masa salarial total
- El 71% de los contratos son Full time
- Software Developer es la categoría con más publicaciones.

## Archivos:
- `Jobs-practica.xlsx` — dataset y dashboard completo.
